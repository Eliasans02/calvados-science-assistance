# Быстрый старт

Краткая инструкция для запуска `dead-regulations-detector`.

## Текущее состояние

После установки зависимостей доступно:

- Streamlit-интерфейс на `http://localhost:8501`
- базовая rule-based детекция
- демо-сценарии для проверки функциональности
- опциональный AI-анализ через выбранного провайдера

## Как запустить

```bash
cd ~/dead-regulations-detector
source venv/bin/activate
streamlit run src/ui/app.py --server.port=8501
```

Откройте:

`http://localhost:8501`

## Базовый сценарий проверки

1. Выберите режим демонстрации в интерфейсе
2. Запустите анализ одного из примеров
3. Проверьте найденные проблемы и объяснения
4. При необходимости экспортируйте результаты

## AI-настройка (опционально)

В боковой панели можно выбрать:

- `openai`
- `claude`
- `github`
- `openrouter`
- `none`

При выборе провайдера, требующего ключ, укажите ключ в UI или через `.env`.

## Что анализируется

Система выявляет:

- устаревшие термины и ссылки
- потенциальные противоречия
- дублирующие формулировки
- потенциально неприменимые нормы
